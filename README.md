## 📌 Professional Overview
This repository serves as a comprehensive portfolio of the analytical pipelines and strategic frameworks developed during my Data Analytics Internship at **Decode Labs**. 

The projects demonstrate a full-stack approach to data science, moving from raw, chaotic data through advanced statistical cleaning, relational SQL extraction and finally into executive-level visual storytelling. 

---

## 📂 Project Directory

### 🧮 [Project 01: Advanced EDA & Feature Engineering](./01_Advanced_EDA/)
**Goal:** Transform chaotic raw data into a mathematically clean dataset ready for machine learning.
* **Skills Applied:** Statistical Imputation (KNN), Outlier Neutralization (IQR Method), Feature Engineering (Temporal/Ratio Extraction).
* **Tech Stack:** Python (`pandas`, `scikit-learn`, `scipy`)

### 📊 [Project 02: Exploratory Data Analysis & Diagnostic Frameworks](./02_Exploratory_Data_Analysis/)
**Goal:** Construct a diagnostic framework to understand the center of gravity, variability, and linear relationships within the dataset.
* **Skills Applied:** Five-Number Summaries, Skewness Diagnosis (Mean vs. Median sensitivity), Pearson Correlation ($r$) Analysis.
* **Tech Stack:** Python (`pandas`, `numpy`, `seaborn`)

### 🗄️ [Project 03: SQL Data Analysis & Relational Logic](./03_SQL_Data_Analysis/)
**Goal:** Translate procedural business questions into declarative queries to filter, group, and aggregate raw data efficiently.
* **Skills Applied:** SQL Execution Order optimization, Row-Level Filtering (`WHERE`), Aggregation (`SUM/COUNT/AVG`), and Bucket Filtering (`HAVING`).
* **Tech Stack:** Python (`sqlite3`)

### 🎨 [Project 04: Strategic Data Visualization & Storytelling](./04_Data_Visualization/)
**Goal:** Synthesize complex data into boardroom-ready visual presentations that drive immediate executive action.
* **Skills Applied:** Data-Ink Ratio Maximization, The Spotlight Rule (Color Theory), Direct Labeling, SCR (Situation-Complication-Resolution) Narrative Framework.
* **Tech Stack:** Python (`matplotlib`, `seaborn`)
